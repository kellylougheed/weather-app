<h1>Weather App</h1>

<b>This weather app was written with JavaScript, jQuery, HTML, and CSS.</b>

The weather app detects the user's location and reports the corresponding weather.

<h2>Features:</h2>

<ul>
<li> The temperature toggles between Fahrenheit and Celsius using jQuery to respond to the user's click and a JavaScript formula that converts between the different units.</li>

<li> Geolocation API approximates the user's location from their IP address.</li>

<li>Weather API uses the results of the Geolocation API to report the weather for the user's location.</li>
</ul>
