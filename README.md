Weather App

This weather app was written with JavaScript, jQuery, HTML, and CSS.

The weather app detects the user's location and reports the corresponding weather.

Features:

The temperature toggles between Fahrenheit and Celsius using jQuery to respond to the user's click and a JavaScript formula that converts between the different units.

Geolocation API approximates the user's location from their IP address.

Weather API uses the results of the Geolocation API to report the weather for the user's location.
